# How to run
### strapi-docker
- run script ./run.sh
### gatsby client
- npm run develop
# How to stop
### strapi-docker
- run script ./stop.sh
### gatsby client
- Just stop process
# Set up
- docker compose
- Before run `docker-compose up`, please unzip db.zip to continue previous state
# Attendtions
- Before push please zip db folder in strapi-docker folder
- Before push please stop all service before commit
